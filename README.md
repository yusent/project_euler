Project Euler
======

These are solutions to the problems in "ProjectEuler+" contest at hackerrank.com, which are an adaptation of the ones at projecteuler.net.

Haskell
------

##### Build:
```
$ ghc --make -o problemX Main.hs
```

##### Run:
```
$ ./problemX
```

##### Unit Testing:
```
$ ghc --make -o test SolutionTest.hs
$ ./test
```

Javascript
------

##### Run:
```
$ node main.js
```

##### Unit Testing:
```
$ mocha test
```
